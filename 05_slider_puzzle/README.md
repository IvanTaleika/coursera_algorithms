# Assignment 5 - Slider Puzzle

This module implements the assignment of the Module 8 of the course. The goal of this task is to utilise Priority Queues
to solve a [slider puzzle](https://en.wikipedia.org/wiki/Sliding_puzzle) - a game when you need to arrange a n*n grid in
order by exchanging a tile with an empty neighbour.

[Full assignment specification](https://coursera.cs.princeton.edu/algs4/assignments/8puzzle/specification.php).
